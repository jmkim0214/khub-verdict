# 찐가판정 — 가격·주장 검증 판정서 (목업)

THE K-HUB / Price-Truth Oracle 의 소비자 표면 목업입니다.
올리브영 상품 URL을 넣으면 12개월 가격 이력과 제품 주장을 검증한 판정서를 보여줍니다.

- **실데이터 판정서**: 실제 수집 데이터(oy.db) 기반 조기 신호. 판정서 번호는 verdicts 대장에 기록됩니다.
- **정식 판정서 미리보기**: 관측 12개월 축적 후의 모습(가상 데이터).

> ⚠️ 조기 신호(관측 12일) 기준의 목업입니다. 브랜드 실명 판정이 포함되어 있어
> 정식 공개(관측 90일 밴드 + 이의제기 절차) 전까지는 링크 공유 수준으로만 사용합니다.
